# codex-agents v0.1.2

## Summary

`v0.1.2` is a cleanup and packaging follow-up release that keeps the `v0.1.1` Codex-native `.toml` agent model, removes non-release docs from the repository, fixes the Chinese README link, and prepares Homebrew packaging to consume the new release artifact.

## Highlights

- keeps the bundled custom agents in Codex-native `agents/*.toml`
- removes historical `docs/plans/` and `docs/specs/` content so `docs/` now contains release notes only
- fixes the Chinese README entry link
- updates Homebrew packaging metadata to use the `v0.1.2` release tarball
- updates the Homebrew formula test to expect `orchestrator.toml`

## User Impact

- source installs and linked installs continue to behave the same as `v0.1.1`
- the bundled command surface remains unchanged
- repository docs are simpler: release notes stay under `docs/releases/`, while historical planning material is no longer shipped in-tree
