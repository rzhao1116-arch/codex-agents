# codex-agents v0.1.3

## Summary

`v0.1.3` is a release-alignment follow-up that brings the repository metadata, release notes, and Homebrew packaging path back into sync with the latest published artifact.

## Highlights

- refreshes release-facing documentation to reflect the current published version
- updates the Chinese README so it no longer describes pre-`v0.1.2` packaging state
- promotes the Homebrew install path to the new `v0.1.3` release artifact
- keeps the Codex-native `.toml` agent bundle and routing CLI surface unchanged

## User Impact

- `brew upgrade rzhao1116-arch/homebrew-tap/codex-agents` points to the `v0.1.3` artifact once the tap update lands
- source installs and linked installs keep the same command surface as `v0.1.2`
- repository docs now match the public install path and latest published release
